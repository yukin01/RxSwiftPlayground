# RxSwiftPlaygrounds

## 環境
cocoapods = 1.4.0  
RxSwift >= 4.0  

## How to Use

bundler はインストール済みとする

```
$ bundle install --path vendor/bundle
$ bundle exec pod --version
$ bundle exec pod install
```

Xcode で一度 project を run すると playground のエラーが消えるはず
